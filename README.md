# Notes Backend API

This is a backend development project built using Node.js, Express.js, and MongoDB.

The project provides REST API functionality for managing notes, including:

- Create Notes
- Update Notes
- Delete Notes
- View Notes

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Features

- RESTful API architecture
- CRUD operations for notes
- MongoDB database integration
- API testing using Postman
- Organized backend folder structure

## API Endpoints

- POST /notes → Create a note
- GET /notes → Get all notes
- PUT /notes/:id → Update a note
- DELETE /notes/:id → Delete a note
